# LEK-Biens Vokter produktrepo

Dette repoet er et rent produktrepo.

## Eier i produktrepoet

- produktkode
- UI/UX
- API-er
- database og migrasjoner
- forretningslogikk
- komponenter
- sider
- et enkelt feature-register for produktflater

## Eier ikke i produktrepoet

- Playwright-tester
- regression-tester
- user journeys
- Aurora-tester
- AKB-tester
- QA-dashboards
- TestLab-scripts
- TestLab-DoD
- testresultater, traces, videoer og screenshots

## Feature-register

- Fil: `product/lek-biens-vokter.features.json`
- Formål: beskrive eksisterende produktflater og hvilke kodeområder de tilhører
- Begrensning: filen skal ikke inneholde testreferanser, user journeys eller QA-logikk

## Testansvar

LEK TestLab er eneste testsystem for LEK-Biens Vokter.
